命令行方式的ss

  • 通过配置文件指定配置信息

  • server填你的服务器ip，sever_port填远程端口号，local_address本地ip，local_part本地端口，password填密码，timeout是延迟时间，method是加密方式，按照实际情况填写并保存

  • 终端运行：sslocal -c ~/.shadowsocks/ss.json

  • 后台运行(不推荐)：sslocal -c ~/.shadowsocks/ss.json -d start

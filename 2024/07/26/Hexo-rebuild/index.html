无法’hexo d’:缺少插件

  • 使用命令：npm install hexo-deployer-git --save (在博客的根目录下执行)

侧边栏文章索引错乱

  • 尽量不要使用跳跃式的标题，比如使用了一个一号标题，然后使用了一个三号标题，接着又使用了一个一号标题，这样很容易就会出现错乱的现象。

  • 标题应当逐级递增逐级递减，这样规范之后才会有清晰美观的显示效果。

hexo文章统计阅读量

  • 使用leanCloud来完成访问统计

  • 需要设置安全链接,参考博客

问题错误码429

  • 出现这个问题的报错信息是too many requests,原因是我们使用leanCloud的免费版本线程数是十分受限的,在我们deploy的时候发送了太多的同步请求导致线程栈溢出而报错.

  • 解决方案 —hexo-leancloud-counter-security 插件 Too many requests 错误
